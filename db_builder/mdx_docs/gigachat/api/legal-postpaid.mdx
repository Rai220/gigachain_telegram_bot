# Работа с API по постоплате {#rabota-s-api-po-postoplate}

Для корпоративных клиентов доступна возможность работы с GigaChat API по постоплатной схеме.

:::note

О том как начать работать с API по предоплатной схеме — в разделе [Быстрый старт для ИП и юридических лиц](/ru/gigachat/legal-quickstart).

:::

Чтобы иметь возможность работать с GigaChat API по постоплатной схеме вам нужно создать проект GigaChat API Enterprise в соответствующем корпоративном пространстве.

Для этого:

1. Зарегистрируйтесь или авторизуйтесь в [личном кабинете](https://developers.sber.ru/studio/login).
2. Перейдите в [корпоративное пространство (постоплата)](/ru/va/about/legal-entities/corporate-space).
3. Нажмите **Создать проект** в левом меню.
4. Выберите **GigaChat API Enterprise** в разделе **AI-модели**.
5. Заполните заявку на получение доступа к GigaChat API по постоплатной схеме.

   Перед отправкой ознакомьтесь с [условиями использования](/ru/policies/gigachat-agreement/corporate-clients-beta).

6. Нажмите кнопку **Отправить заявку**.

:::tip

Если у вас уже есть личный кабинет, авторизуйтесь и отправьте заявку на доступ к GigaChat API по [ссылке](https://developers.sber.ru/studio/workspaces/my-corp-space?billing_type=postpaid).

:::

После отправки запроса на доступ к GigaChat API вам нужно сообщить о согласии с офертой. Для этого:

1. Скачайте и заполните [заявление](pathname:///files/policies/gigachat/application-offer-gigachat-postpaid.docx). 

   :::note

   Чтобы заполнить поле **Лицевой счет**, скопируйте его ID в правом верхнем углу окна личного кабинета или в левом углу на главной странице личного кабинета:

   <Image img={require('../../va/about/legal-entities/images/workspace-parameters.png')} width={250} alt="Обязательный текст" />

   ID лицевого счета представлен в формате `LBO-хххххххх-хххх-хххх-хххх-хххххххххх`.

   :::

2. Отправьте нам заполненное заявление в формате `doc/docx` по [ЭДО](/ru/va/about/legal-entities/edo).

Рассмотрение запроса на доступ обычно занимает до двух рабочих дней после предоставления документов.
Вы получите письмо на свой электронный адрес с положительным или отрицательным ответом. 
Чтобы сократить время рассмотрения заявки, дополнительно отправьте письмо на <a href="mailto:developer@sberbank.ru?subject=GigaChat.%20Заявление%20для%20пространства%20<ID%20вашего%20пространства>">электронную почту</a>.


Дальнейшая работа с API аналогична [общему процессу](/ru/gigachat/legal-quickstart#shag-4-poluchenie-tokena-dostupa).
