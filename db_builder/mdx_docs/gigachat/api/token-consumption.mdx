# Мониторинг статистики потребления GigaChat API {#monitoring-potrebleniya-giga-chat-api}

Мониторинг потребления доступен только в проектах для ИП и юридических лиц ([`scope=GIGACHAT_API_CORP`](/ru/gigachat/legal-quickstart#shag-4-poluchenie-tokena-dostupa)).

Для просмотра данных о потреблении токенов GigaChat API:

1. [Авторизуйтесь в личном кабинете](https://developers.sber.ru/studio/login).
2. Перейдите в проект **GigaChat API**.

В интерфейсе проекта доступны график и таблица с данными об использованных услугах: генерации с помощью различных моделей, создании эмбеддингов, а также данные о количестве затраченных сессий и токенов.

<Image img={require('./images/tokens-consumption-monitoring.png')} width={600} alt="Мониторинг потребления токенов в интерфейсе проекта" />

Для скачивания отчета в формате xlsx, нажмите кнопку **Скачайте отчет** в верхнем правом углу.
